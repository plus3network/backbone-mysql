# backbone-mysql

A sync module for Backbone.js and Node.js for use with MySQL

## Getting Started
Install the module with: `npm install backbone-mysql`

```javascript
var backbone_mysql = require('backbone-mysql');
backbone_mysql.awesome(); // "awesome"
```
